# 3. 실습용 백엔드 서버 세팅

* 의존성 추가 : 모든 패키지와 의존성 설치
```
> npm i
또는
> npm install
```

- Supabase DB 초기화
```
> npx prisma db push
```

- 초기 데이터 자동으로 삽입 : 자동으로 seed데이터 즉, 데이터 기초 데이터들이 자동으로 설정
```
> npm run seed
```

- 백엔드 서버 빌드
```
> npm run build
```

- 백엔드 서버 실행 → https://localhost:12345
```
> npm run start
```

- 가동 중인 서버 끄기
```
> ctrl + c
```

- 사용 중인 데이터를 조회 : 사용중인 데이터를 조회할 수 있는 별도의 페이지로 이동 → https://localhost:5555
```
> npx prisma studio
```
