# 1. Next.js 를 소개
- Next.js는 React.js 전용 웹 개발 `Framework`
- React.js를 보다 더 강력하고 편하게 사용할 수 있는 기능들을 제공함 → 페이지 라우팅, 빌트인 최적화 기능, 다이나믹 HTML 스트리밍 등...

### ※ Next.js vs React.js
- Next.js
  - React.js 전용의 웹 개발 `Framework`
  - 기능 주도권이 누구에게 있는가 → 주도권이 개발자에게 X → 프레임워크가 제공하는 기능을 이용하거나 허용하는 범위 내에서만 추가 도구 사용 가능 → 자유도 낮음
  - ex) Next.js, Remix ... 

- React.js
  - UI개발을 위한 JavaScript `Library`
  - 기능 주도권이 누구에게 있는가 → 주도권이 개발자에게 O → 기능 구현을 원하는 방향으로 진행, 쓰고싶은 도구와 기술을 사용 → 자유도 높음
  - React.js, JQuery ...
