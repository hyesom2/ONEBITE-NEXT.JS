# 4. 스켈레톤 UI 적용하기
- 뼈대 역할을 하는 UI
- 페이지에 일부 컨텐츠가 로딩이 되는 동안 실제 컨텐츠 대신 컨텐츠와 비슷하게 생긴 실루엣 정도만 보여주는 것
- 사용자는 로딩 시간 동안 컨텐츠가 로딩이 끝났을 때 컨텐츠 예상 가능 → 더 좋은 사용자 경험 제공

---

### 예&#41; book-item의 skeleton UI 만들기
- 폴더 구조
```
components
 ┣ skeleton
 ┃ ┣ book-item-skeleton.module.css
 ┃ ┣ book-item-skeleton.tsx
 ┃ ┗ book-list-skeleton.tsx
 ┣ book-item.module.css
 ┣ book-item.tsx
 ...
```

- 컴포넌트 파일들
```tsx
import type { BookData } from '@/types';
import Link from 'next/link';
import style from '@/components/book-item.module.css';

export default function BookItem({ ... }: BookData) {
  return (
    <Link href={`/book/${id}`} className={style.container}>
      <img src={coverImgUrl} />
      <div>
        <div className={style.title}>{title}</div>
        <div className={style.subTitle}>{subTitle}</div>
        <br />
        <div className={style.author}>
          {author} | {publisher}
        </div>
      </div>
    </Link>
  )
}
```
```tsx
// book-item-skeleton.tsx

import style from './book-item-skeleton.module.css';

export default function BookItemSkeleton() {
  return (
    <div className={style.container}>
      <div className={style.cover_img}></div>
      <div className={style.info_container}>
        <div className={style.title}></div>
        <div className={style.subTitle}></div>
        <br />
        <div className={style.author}></div>
      </div>
    </div>
  )
}
```
```css
// book-item-skeleton.module.css

.container {
  display: flex;
  gap: 15px;
  padding: 20px 10px;
  border-bottom: 1px solid rgb(220, 220, 220);
}

.cover_img {
  width: 80px;
  height: 105px;
  background-color: rgba(230, 230, 230);
}

.info_container {
  flex: 1;
}

.title, .subTitle, .author {
  width: 100%;
  height: 20px;
  background-color: rgba(230, 230, 230);
}
```
1. book-item-skeleton.tsx 파일이 book-item.tsx 파일의 컴포넌트와 `동일한 구조`를 갖도록 div 태그들을 배치
2. 각 구조에 css를 적용
3. 만들어진 book-item-skeleton 컴포넌트를 `Suspense의 fallback Props`에 설정
