# 6. 스타일링
- Next.js 앱의 스타일링은 사실상 `리액트`의 컴포넌트 스타일을 설정하는 과정과 동일

### ① 인라인 스타일
```tsx
// 예)
export default function Home() {
  return <h1 style={{ color: red }}></h1>
}

```

```tsx
import './index.css';

export default function Home() {
  return ...
}
```


### ② CSS Modules
- Next.js는 `.module.css` 확장자를 사용하여 CSS Modules를 기본적으로 지원
- 충돌을 걱정하지 않고 다른 파일에서 동일한 클래스 이름을 사용
- 이러한 동작은 CSS Modules를 컴포넌트 수준의 CSS를 포함하는 이상적인 방법으로 만든다.

```tsx
// index.tsx
import styles from './index.module.css';

export default function Home() {
  return <h1 className={styles.h1}>index 페이지</h1>
}
```
```css
/* index.module.css */
.h1 {
  color: red;
}
```
