3. searchbar.tsx
- `useRouter`
  - next/navigation으로부터 불러오는 Next.js가 제공하는 자체적인 Hook
```tsx
'use client';

import { useRouter } from 'next/navigation';

export default function Searchbar() {
  const router = useRouter();
  ...
}
```

- `useSearchParams`
  - next/navigation으로부터 불러오는 Next.js가 제공하는 자체적인 Hook
  - 현재 페이지에 전달된 `query string의 값`을 꺼내올 수 있는 기능을 제공
  - searchParams 객체의 `get 메서드`를 활용
```tsx
'use client';

import { useSearchParams } from 'next/navigation';

export default function Searchbar() {
  const searchParams = useSearchParams();

  const q = searchParams.get('q');

  return (
    ...
  )
}
```
```
참고
- Page Router 버전에서 query string을 불러올 때 router.query.q식으로 불렀지만 App Router 버전에서는 불가능
- App Router 버전 부터는 router객체에 query property가 제공되질 않음 
```
