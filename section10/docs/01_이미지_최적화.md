# 1. 이미지 최적화
- 웹 페이지에서 평균적으로 가장 많은 용량을 차지하는 요소

### ※ 다양한 이미지 최적화 기법들
- webp, AVIF 등의 차세대 형식으로 변환
- 디바이스 사이즈에 맞는 이미지 불러오기
- 레이지 로딩 적용
- 블러 이미지 활용 <br />
→ **Next.js는 대부분이 최적화 기법들을 자체적으로 제공**

### ※ Image 컴포넌트
예&#41;
```tsx
import Image from 'next/image';

export default function Page() {
  return (
    <Image
      src="profile.png"
      width={500}
      height={500}
      alt="Picture of the author"
    />
  );
}
```
